<h1 align="center">
  whyframe
</h1>

`whyframe` allows rendering any markup within an `iframe` when writing with your UI framework, e.g. Svelte, Vue, Solid, Preact, and React. Simply:

<!-- prettier-ignore -->
```html
<iframe data-why>
  Hello world!
</iframe>
```
